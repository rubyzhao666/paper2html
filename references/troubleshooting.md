## PDF 解析相关

### Q: PDF 解析失败，返回空白内容

**可能原因**:
1. PDF 是扫描版（图片型），没有可提取的文本层
2. PDF 加密或受保护
3. PDF 文件损坏

**解决方案**:
- 使用可编辑的 PDF（非扫描版）
- 尝试用工具（如 Calibre）转换扫描版 PDF
- 检查 PDF 是否可以正常打开

### Q: 公式识别不完整或错误

**可能原因**:
1. 使用的是 v2 模型，公式识别能力有限
2. PDF 中公式图片化，无法提取

**解决方案**:
- 使用 v3-2026 模型（`model="v3-2026"`）
- 对于复杂公式，可能需要手动校准

### Q: 图片未提取或提取不完整

**可能原因**:
1. PDF 中的图片嵌入方式特殊
2. 图片在表格或文本框中
3. 彩色图片被压缩

**解决方案**:
- 检查输出目录的 images 文件夹
- 尝试使用 Adobe 等工具预处理 PDF
- 查看原始 PDF 确认图片是否存在

### Q: 表格结构混乱

**可能原因**:
1. PDF 中表格使用了合并单元格
2. 表格跨页

**解决方案**:
- 检查 Markdown 中的表格语法
- 可能需要手动调整表格格式

### Q: arXiv 下载失败

**可能原因**:
1. 网络连接问题
2. arXiv 服务暂时不可用
3. 论文 ID 不存在

**解决方案**:
- 检查网络连接
- 稍后重试
- 确认 arXiv ID 正确（如 `2402.19473`）

## AI 理解相关

### Q: 理解层返回空结果

**可能原因**:
1. LLM API Key 未设置或无效
2. API 调用失败
3. Markdown 内容为空

**解决方案**:
- 设置 `DEEPSEEK_API_KEY` 环境变量
- 检查 API Key 是否有效
- 使用 `--debug` 查看详细错误日志
- 使用 `--no-understand` 跳过理解层

### Q: TL;DR 生成失败

**可能原因**:
1. 摘要内容为空或太短
2. LLM API 调用超时

**解决方案**:
- 确保 PDF 解析成功，有摘要内容
- 增加 LLM 超时时间
- 检查网络连接

### Q: 章节摘要不完整

**可能原因**:
1. 论文结构不规范
2. LLM 并发调用受限

**解决方案**:
- 部分失败不影响整体，缺失部分会用默认值
- 检查 `--debug` 日志

### Q: Mermaid 架构图未生成

**可能原因**:
1. LLM 未成功生成图代码
2. Mermaid 语法错误

**解决方案**:
- 检查理解层输出的 `mermaid_code` 字段
- 手动修复 Mermaid 语法错误
- 生成失败时，HTML 中会显示占位符

## HTML 渲染相关

### Q: 生成的 HTML 文件为空或错误

**可能原因**:
1. 解析层失败
2. 渲染层代码错误

**解决方案**:
- 使用 `--debug` 查看详细日志
- 检查 Markdown 内容是否为空
- 确认输出路径有写权限

### Q: 公式未渲染

**可能原因**:
1. KaTeX CDN 加载失败
2. 公式语法错误

**解决方案**:
- 检查网络连接
- 确认公式使用 `$...$` 或 `$$...$$` 语法
- 离线查看可使用本地 KaTeX

### Q: 图片不显示

**可能原因**:
1. 图片路径错误
2. Base64 编码失败
3. 图片格式不支持

**解决方案**:
- 检查解析层输出的图片列表
- 确认图片文件存在
- 支持格式：jpg, jpeg, png, gif, webp

### Q: 主题切换无效

**可能原因**:
1. JavaScript 被禁用
2. CSS 选择器冲突

**解决方案**:
- 确保浏览器启用了 JavaScript
- 检查 HTML 中的 class 名称

### Q: Mermaid 图不渲染

**可能原因**:
1. Mermaid CDN 加载失败
2. Mermaid 代码语法错误
3. JavaScript 被禁用

**解决方案**:
- 检查网络连接
- 验证 Mermaid 代码语法
- 确保浏览器启用了 JavaScript

## API 相关

### Q: 401 Unauthorized

**原因**: API Key 无效或过期

**解决方案**:
1. 检查 API Key 是否正确
2. 确认 API Key 是否有效（前往对应平台检查）
3. 确认环境变量已正确设置

### Q: 403 Quota Exceeded

**原因**: API 调用额度用完

**解决方案**:
1. 等待额度重置（通常每月）
2. 购买付费套餐
3. 使用本地 PPX 作为备选

### Q: 429 Rate Limit

**原因**: 请求过于频繁

**解决方案**:
1. 添加请求间隔（至少 1 秒）
2. 减少并发请求
3. 升级套餐获取更高限流

### Q: 超时错误

**原因**: 解析大文件或网络不稳定

**解决方案**:
1. 增加超时时间
2. 尝试分页处理大 PDF
3. 检查网络连接
4. 使用本地 PPX

### Q: 网络连接错误

**可能原因**:
1. 网络不稳定
2. 防火墙/代理拦截
3. API 服务不可用

**解决方案**:
1. 检查网络连接
2. 配置代理（如果需要）
3. 查看 API 服务状态
4. 使用本地 PPX

## 本地 PPX 相关

### Q: ppx 命令找不到

**原因**: 未正确安装或 PATH 未配置

**解决方案**:
```bash
pip install memect-ppx
ppx install
```

### Q: ppx 解析失败

**可能原因**:
1. Python 版本不兼容（需要 3.12+）
2. 依赖缺失

**解决方案**:
- 检查 Python 版本：`python --version`
- 升级 Python 到 3.12+
- 安装依赖：`pip install -r requirements.txt`

## 命令行使用相关

### Q: 提示 "No module named 'paper2html'"

**原因**: 未在正确目录执行，或包未安装

**解决方案**:
```bash
cd paper2html
python paper2html.py --help
```

### Q: 提示 "DOC2X_APIKEY 未设置"

**原因**: 环境变量未配置

**解决方案**:
```bash
export DOC2X_APIKEY="your-api-key"
# 或使用命令行参数
python paper2html.py --api-key "your-api-key"
```

### Q: 输出文件路径不对

**原因**: 未指定输出路径或路径问题

**解决方案**:
```bash
# 指定绝对路径
python paper2html.py --input paper.pdf --output /full/path/to/output.html

# 指定相对路径
python paper2html.py --input paper.pdf --output ./output/my_paper.html
```

### Q: 如何跳过 AI 理解层

**原因**: LLM API 额度不足或不需要摘要

**解决方案**:
```bash
python paper2html.py --input paper.pdf --no-understand
```

### Q: 如何查看详细日志

**原因**: 需要排查问题

**解决方案**:
```bash
python paper2html.py --input paper.pdf --debug
```
